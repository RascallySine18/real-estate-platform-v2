# Real Estate Platform

**Real Estate Platform** — десктопное приложение для Windows, предназначенное для управления объявлениями о продаже и аренде квартир. Позволяет регистрировать пользователей, добавлять объявления, просматривать фото, контакты владельцев и бронирования. Разработано в рамках учебного проекта с использованием Python и PyQt5 для удобного интерфейса.

Приложение использует базу данных SQLite для хранения данных о пользователях, квартирах и объявлениях. Поддерживает роль администратора для управления контентом.

---

## Возможности

- Регистрация и вход пользователей (администратор/пользователь)
- Добавление, редактирование и удаление объявлений о продаже/аренде квартир (цена, площадь, комнаты, адрес, этаж, описание, фото)
- Просмотр списка объявлений с деталями и фото (включая просмотрщик с навигацией)
- Поиск объявлений по критериям (цена, комнаты, этаж)
- Управление контактами владельцев (телефон, email)
- Административная панель для модерации объявлений
- Адаптивный интерфейс с изменением размеров элементов

## Установка и запуск

### Требования
- Python 3.8+
- Установленный PyQt5 (`pip install PyQt5`)
- Windows 10+

### Шаги по установке

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/RascallySine18/real-estate-platform-v2.git
2. Установите зависимости:
   ```bash
   pip install PyQt5
3. Запустите приложение:
   ```bash
   python main.py
